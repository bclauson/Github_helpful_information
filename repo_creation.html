Here is how to create a git repo!

  1. Open git bash
  2. cd to where you want to create your repo (I reccomend desktop -> coding folder -> repo
  3. create repo by: mkdir (repo name)
  4. cd into repo
  5. create a file to work with: touch file_name.html
  6. create a css folder, image folder, and whatever other folders you want: mkdir css img
  7. add other files such as css within the css folder: touch css/styles.css
  8. Initialize git repo: git init
  9. Add current elements to repo: git add .
  10. commit changes: git commit -m "initialized repo"
  11. go to your github in browser and create a repo for this project
  12. connect repo to repo in github: git remote add (initials) (github link)
  13. After this step you might have to enter password or PAT
  14. push to github: git push (initials) main

Now you should have a repository connected to github! Remeber to add and commit as you make changes

after adding changes enter this into git bash

  1. cd to repo if you are not already
  2. git add . or git add (specific file)
  3. git commit -m "I made this specific change"
  4. *WHEN DONE WORKING* git push (initials) main
